## Nilai Truthy dan Falsy
Truthy dan Falsy adalah sebuah sebutan bagi nilai yang dianggap true atau false saat diuji dalam konteks boolean meskipun bukan bertipe boolean.

Falsy merupakan nilai yang dianggap false saat dipakai di if, while, dan operator logika seperti &&, dan ||.
contoh nilai falsy (selain nilai dibawah ini adalah nilai Truthy): 
    1. false
    2. 0
    3. -0
    4. 0n
    5. ""
    6. null
    7. undefined
    8. NaN

Truthy merupakan nilai yang dianggap true saaat diuji sebagai boolean, seperti:
    1. true
    2. 1
    3. {}
    4. []
    5. " "
